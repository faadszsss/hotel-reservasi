# Sistem Manajemen Kamar Hotel (Reservasi)

Project ini dibuat menggunakan PHP & MySQL.

## Fitur
- Registrasi Manajer Hotel (email sebagai username)
- Aktivasi akun via token (24 jam)
- Login & Logout
- CRUD Data Kamar
- Lupa Password
- Dashboard Manajer

## Struktur Database
Terdapat 2 tabel:
- users
- rooms
